# CarManager – Spring Boot Project

A simple REST API for managing cars, built with Spring Boot.  
The application implements full CRUD functionality and uses Spring Data JPA for database interaction.

Cars are stored in an H2 database with persistent storage, and each entity includes attributes such as brand, model, price, and engine type (enum).

The project also includes a minimal frontend that consumes the API using basic HTTP requests.

# MovieTracker – Spring Boot Project

A REST API for tracking movies, developed using Spring Boot.  
It supports CRUD operations and allows managing movies based on status (e.g. watched, planned) and rating.

The application uses Spring Data JPA for database access and H2 for persistence.  
It demonstrates basic backend structure and data handling in a clean and modular way.
